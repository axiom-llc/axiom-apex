# APEX Blueprints & Examples

This directory contains production-ready implementation blueprints and script collections demonstrating the operational capabilities of the APEX runtime [2].

## Directory Content

*   `cli_demos.sh` — A unified command-line entry script organizing swarm tasks, iterative fix loops, and research agent executions [2].
*   `logistics-dashboard/` — A full-stack data ingestion pipeline with a compliant dashboard tracking automated shipment verification [2].
*   `voice-agent/` — A Twilio-integrated telephony automation script leveraging Gemini's real-time execution pipelines [2].
*   `test_logistics.py` — Local compliance and pipeline regressions [2].

---

## Run Guidelines

To run the core CLI demonstration harness:
```bash
chmod +x cli_demos.sh
./cli_demos.sh
```
